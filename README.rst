Twitter scraper
===============

Requirements
------------

Run

.. code::

    pip install -r requirements.txt

This installs the following libraries:

* matplotlib
* twitter-scraper

Usage
-----

.. code::

    python twitter.py USERNAME
    python twitter.py elonmusk
    python twitter.py JeffBezoz
    ...
